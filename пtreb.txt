Требования:

Для работы программы необходимы:
Аппаратные средства:
мобильное устройство:
 сенсорный экран, 
процессор ?
оперативная память 1Гб и более
Внутренняя память 4Гб и более
Звук
Вибрация

Программное обеспечение: 
Android v5 и старше
TalkBack для Android

Требования к разработке:

ПО для разработки:
Android Studio под Windows
AIDE JAVA под Android

Языки, используемые при разработке:
JAVA, XML, SQL.

Требования к функциональным возможностям программы:
- создание набора фигур с замкнутым контуром,
- реализация сеанса игры - взаимодействие с пользователем по распознаванию фигуры, 
- сохранение, вывод и очистка результатов игры (достижений):
 идентификатор фигуры и время распознавания,
- обеспечение корректного отображения всех окон при вертикальной и горизонтальной ориентации экрана,
- возможность изменения параметров: цветовая схема, толщина контура фигуры, способы отклика (звук и вибрация/только звук/только вибрация),
В диалоговой панели Game предусмотреть игровое поле и 2 кнопки: Угадал(а) и Открыть фигуру.
В диалоговой панели Answer предусмотреть таблицу с вариантами ответов, группой кнопок с зависимой фиксацией, кнопками Отправить и Дотмена.
В диалоговой панели User предусмотреть таблицу для  таблицу достижений и кнопку Ок.
В диалоговой панели Settings предусмотреть
раскрывающийся список для выбора уровея сложности, кнопку Загрузить для 
обновления базы и кнопку "Отправить.

- Все используемые диалоговые элементы должны корректно озвучиваться программой экранного доступа TalkBack.

При реализации диадога на нескольких языках (в настройки добавляется выбор языка пользовательского интерфейса).).
